This is LRCGET — a desktop music player with synced lyrics support.

Stack: Tauri v2 (Rust backend + Vue 3 frontend).

For project structure:

- Frontend Vue part: see `src/ARCHITECTURE.md` file.
- Backend Rust part: see `src-tauri/ARCHITECTURE.md` file.

After completing changes, run `cargo check` and update `ARCHITECTURE.md` files as needed.
